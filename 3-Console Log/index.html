What is the Console?

The console is a built-in tool in web browsers that allows developers to view messages, run JavaScript code, and debug their applications. It does not show anything on the webpage ‚Äî only inside DevTools.

To open the console:

  • Press F12 on your keyboard.
  • Or right-click on the webpage and select "Inspect" or "Inspect Element."
  • Then, navigate to the "Console" tab.
  • Or ctrl + shift + j

The console displays messages logged by the console.log() function, as well as errors and warnings.

What is console.log()?

console.log() is a JavaScript function that prints messages to the console. It is commonly used for debugging and testing code.

Example:

console.log("JS file is connected!");

When this code runs, "JS file is connected!" will appear in the console.

Examples logging different data types:

  • console.log("Hello world") //Logs a string
  • console.log(42); // Logs a number
  • console.log(true); // Logs a boolean
  • console.log("Alice is", 30, "years old"); // Logs multiple values
  • console.log(name); // Logs a variable
  • console.log(6 + 8); // Logs a mathematic operation

üêû Debugging Errors in Your Script File

If there are errors in your JavaScript code, they will be displayed in the console. Common errors include syntax errors, reference errors, and type errors.

Example of a Reference Error:

console.log(undeclaredVariable);

This will produce a ReferenceError in the console because undeclaredVariable has not been defined.

To fix errors, check the error message in the console and review your code for mistakes.
