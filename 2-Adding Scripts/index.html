Adding Scripts to HTML

üí°In this lesson, we will learn four methods for adding JavaScript files to an HTML document using the <script> tag.

  • An html file can have multiple script tags.
  • Scripts can be written directly inside the HTML file or can be linked as external files.
  • Internal and external scripts can be placed in the <head> or <body> tags.
  • Code for internal scripts is placed inside script tags : <script> </script>
  • Code for external scripts is placed inside an external file and linked using the src attribute of the: <script src="myScript.js"></script>

‚úîÔ∏è Best Practice

It is best to use external scripts and place them in the </head> tag.

  • Best practice for production
  • Reusable across pages
  • Cached by the browser (much faster)
  • Cleaner code structure (HTML, CSS, JS separated)
  • Easier to debug
  • Works better with frameworks, version control, and deployment
