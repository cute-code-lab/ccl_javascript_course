‚ö° What Can JavaScript Do?

JavaScript is the programming language of the web. It helps websites come alive by calculating data, updating content, and responding to user actions. With JavaScript, we can change HTML, update CSS, and control how elements appear on the page.

1. Change HTML content:

For example, we can select an element using getElementById() and update its text. 

document.getElementById("change-html").innerHTML = 2;

2. Change HTML attributes:

Try switching the src of an image to show something different.

3. Change styles (CSS):

JavaScript can adjust fonts, colors, spacing, and more.

document.getElementById("demo").style.fontSize = "35px";

4. Hide & Show Elements:
